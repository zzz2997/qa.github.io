HTML 文档 = 网页

  • HTML 文档描述网页
  • HTML 文档包含 HTML 标签和纯文本
  • HTML 文档也被称为网页

Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容：

例子解释

  • 与 之间的文本描述网页

  • 与

    之间的文本是可见的页面内容

  • 与

    之间的文本被显示为标题

  • <p> 与 </p> 之间的文本被显示为段落
